## Frequently Asked Questions
---
* Q: I don't know how to code!
* A: You don't need to know anything about coding to write a markdown page. Markdown is made to be easy. You can watch [this](https://youtu.be/_PPWWRV6gbA?t=25) crash course on markdown to know how to use it. **If you absolutely refuse to learn markdown, you can write a page in regular english and send it to us to make it look fancy.**

* Q: Is the 6b6t wiki biased?
* A: This wiki aims to be as unbiased as possible. If you think something is biased or a change needs to be made, please contact us or make a pull request.

* Q: Is the 6b6t wiki accurate?
* A: Again, we try our best but cannot be 100% perfect. The wiki is **totally open for anyone to suggest improvements and edit pages**.

* Q: How can I contribute to the 6b6t wiki?
* A: You can fork the GitHub repository, modify it then open a pull request which might be accepted.

* Q: How do i fork a repo? How do i modify it? How do i open a pull request?
* A: Forking is easy! Just press the fork button at the top right. Then, to modify it go to the docs folder, choose your subfolder(groups/players/history), then open your .md page and hit the pencil icon to start editing. Press commit changes when you are done, then to make a pull request watch this tutorial https://www.youtube.com/watch?v=nCKdihvneS0

## About 6b6t
---
6b6t is a Minecraft anarchy server, running on modified Folia server software that, along with other optimizations, allows 1000 players to be online at the same time in the same world world. Just like other anarchy servers, it places an emphasis on player freedom, but unlike most other anarchy servers it has /tpa and /home, the reasons for this are explained [here](https://blog.6b6t.org/how-tpa-is-great-for-anarchy/). 6b6t is owned by [Qbasty](./Players/qbasty.md) and was started on January 16th, 2022, and has merged maps with 7b7t. It was created with the hopes of being the largest lag-free anarchy server with no queue.
